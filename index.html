วิธีเล่น Possession
1
⚔️ โจมตี
ใช้แต้ม ATK ในการโจมตี ยิ่งใช้มาก ดาเมจยิ่งสูง
2
🛡️ ป้องกัน
ใช้แต้ม DEF รับดาเมจ — ป้องกัน 1 DEF ต่อดาเมจ 1 หน่วย
3
🚶 เดิน
ใช้แต้ม RUN เดิน — 1 RUN ต่อ 1 ช่อง
4
✨ สกิล
ใช้แต้ม SPELL ในการปลดปล่อยสกิล แต่ละสกิลมีต้นทุน SPELL ต่างกัน
5
🎲 ระบบแต้ม
ทอยเต๋าต้นเทิร์นเพื่อรับแต้ม ATK · DEF · RUN · SPELL แล้วจัดสรรตามกลยุทธ์
6
🏰 โจมตีป้อม
ใช้ ATK โจมตีป้อมของคู่ต่อสู้ให้เหลือ 0 เพื่อชนะ
7
🏆 เงื่อนไขชนะ
ทำลาย ป้อมของคู่ต่อสู้ ให้เหลือ HP = 0 เพื่อคว้าชัยชนะ!
